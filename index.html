GitHub Pages × Tawk.to 測試

這是一個最小可用的靜態網站，右下角會載入 Tawk.to 聊天元件。

開始測試

  1. 把這個專案上傳到 GitHub。
  2. 在 GitHub 的 Settings → Pages 啟用 Pages。
  3. 把下方的 YOUR_TAWK_WIDGET_ID 換成你自己的 widget id。
  4. 重新整理網站頁面，應會看到右下角的聊天按鈕。

若你暫時沒有 widget id，也可以先保留預設，網站仍可正常顯示。

常見問題

如何取得 Tawk.to widget id？

登入 Tawk.to 後台 → Administration → Chat Widget → Install Widget，複製腳本中 https://embed.tawk.to/<你的 widget id>/default 的那段 ID。

GitHub Pages 網址是什麼？

個人網站：https://<你的帳號>.github.io/
專案網站：https://<你的帳號>.github.io/<repo-name>/
